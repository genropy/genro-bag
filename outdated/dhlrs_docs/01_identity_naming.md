
# Global Identity & Naming Scheme

Each HLRS node has a global identity:
    <node_path>.hlrs.<domain>

Examples:
    hlrs.softwell.it
    alfa.beta.hlrs.softwell.it
    profiles.eu.cluster.hlrs.softwell.it

Browser sessions add a session suffix:
    <hlrs_node>@<page_id>.<local_path>

Example:
    alfa.beta.hlrs.softwell.it@page_23af.ui.form.name
